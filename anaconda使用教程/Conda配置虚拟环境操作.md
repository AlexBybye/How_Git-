

**1****：安装****anaconda****，配置好路径**  
**下述操作在****anaconda prompt****终端实现。**

**2****：创建环境**  
conda create --prefix 本地绝对路径\ python版本 e.g. D:\conda\envs\虚拟环境名\ python=python3.10.6 (选python版本）

**3.****查看环境与切换退出环境**  
conda env list  
*表示我们目前所在环境
 
conda activate 要切换的环境的路径
 
conda deactivate 则退出
 
conda env remove -p 要删除的环境的路径
